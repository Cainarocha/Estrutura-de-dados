# 📊 Estruturas de Dados em Java (Fila, Lista e Pilha)

## 📖 Descrição
Este projeto tem como objetivo implementar e demonstrar o funcionamento de estruturas de dados fundamentais utilizando **Java**.  
Foram desenvolvidas implementações de **Fila (Queue)**, **Lista (List)** e **Pilha (Stack)**, aplicando conceitos essenciais de **Estrutura de Dados** e **Programação Orientada a Objetos (POO)**.

---

## 🎯 Objetivos
- Compreender o funcionamento das estruturas de dados básicas  
- Implementar manualmente estruturas sem uso de bibliotecas prontas  
- Aplicar conceitos de POO na organização do código  
- Desenvolver lógica de manipulação de dados  

---

## 🧱 Estruturas Implementadas

### 🔹 Pilha (Stack)
Estrutura do tipo **LIFO (Last In, First Out)**:
- Inserção: `push()`
- Remoção: `pop()`
- Consulta do topo: `peek()`

### 🔹 Fila (Queue)
Estrutura do tipo **FIFO (First In, First Out)**:
- Inserção: `enqueue()`
- Remoção: `dequeue()`
- Consulta do início: `front()`

### 🔹 Lista (List)
Estrutura dinâmica para armazenamento de elementos:
- Inserção de elementos  
- Remoção de elementos  
- Busca de elementos  
- Exibição da lista  

---

## 🐒 Funcionalidades
- Inserir elementos nas estruturas  
- Remover elementos  
- Visualizar os dados armazenados  
- Testar o comportamento de cada estrutura  
- Interação via terminal  

---

## 🧬 Conceitos Utilizados
- Estruturas de Dados  
- Programação Orientada a Objetos (POO)  
- Encapsulamento  
- Manipulação de Arrays ou Nós (encadeamento)
